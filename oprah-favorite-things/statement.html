Oprah has three favorite things - apples, bananas and flat screen TVs (in that order). Ellen, the generous, gave her a gift certificate worth dollars valid for a single-receipt purchase at their favorite store.

Oprah canvasses the prices. She discovers that one apple is worth dollars, one banana is worth dollars and one flat screen TV is worth dollars. Moreover, she wants to buy at least one of each! Furthermore, so that no dollar is wasted, she wishes to spend exactly dollars for her single purchase.

Your task is this: List down how many of each item Oprah should buy. If there are no solutions, you should say so. If there are multiple solutions, you should list them all. These are elaborated on the output format section of this problem.

Input Format
The input contains exactly two lines.
The first line contains a single integer, , which is the value (in dollars) of the gift certificate obtained from Ellen, the generous.
The next line contains exactly three integers , , and , separated by single spaces. , and are the price per unit of an apple, a banana and a flat screen TV, respectively.

Output Format
If it is impossible to spend exactly dollars given the prices, output a single line IMPOSSIBLE.

Otherwise, for each solution, output a line describing how many of each item to buy. Each line must contain exactly three space-separated integers , , (in that order) where is the number of apples she should buy, is the number of bananas she should buy and is the number of flat screen TVs she should buy.

If there are multiple solutions, output first the solutions which let her buy the most number of apples, Oprah's favorite thing. If there are multiple solutions with equal (number of apples), then output first the solutions which let her buy the most number of bananas, Oprah's second favorite thing.

Constraints




It is guaranteed that the number of solutions per test case is at most .

499
3 4 450

Sample Output 1

Explanation
In the first case, there are four ways to make a purchase worth 499 dollars. Each of them is listed in the correct order described in the output format.

In the second case, all the prices are even. Hence, there is no way to spend exactly 499 dollars.

The third case shows (more clearly) how to properly sort your answers. As you can see, the solution with is printed first. For the solutions 2 2 1 and 2 1 2, the former should be printed first as it represents a larger -value. And so on.
