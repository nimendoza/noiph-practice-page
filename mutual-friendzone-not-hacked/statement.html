On the strangely named social networking site Acquaintancester, a user can have friends. Moreover, the only links on one's profile page are links to the profiles of that person's friends. Clicking on any of these links brings you to the profile whose name you clicked.

A group of people have accounts on Acquaintancester. Each person has at least one friend within the group of . The friendship network of these people has a rather peculiar property though.

  1. Starting from anyone's profile page, it is possible to reach anyone else's profile page (by clicking the available links).

  2. If anyone unfriends any one of their friends, the first condition will not be true anymore.

Two different people are said to be in each other's friendzone if it is possible to reach each other's profile with exactly one click.

Two different people are said to be in each other's mutual friendzone if it is possible to reach each other's profile with exactly two clicks.

How many pairs of people are in each other's friendzone and how many pairs of people are in each other's mutual friendzone?
