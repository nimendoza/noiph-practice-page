Two friends, X and O, play a game of xoxo. The game is played on an grid, initially empty. The players take turns placing X's and O's in empty cells in the grid. The game ends when there are no more empty cells in the grid.

A point is scored for X for every unique set of four X's consecutively lined up horizontally, vertically or diagonally. A point is scored for O for every unique set of four O's consecutively lined up horizontally, vertically or diagonally.

Suppose you are given an grid representing a finished xoxo game. Your task is to determine whether X or O has more points and how many more points that player has than the other.

Note that X always plays first.

Input Format
The first line of input contains the integers and , the number of rows and the number of columns, respectively. The next lines contain strings of length consisting only of X's and O's.

Output Format
If it is an invalid grid (i.e. it is impossible to reach such a grid), output INVALID.

If it is a tie, output BOTH PLAYERS ARE TIED AT p POINT(S), where is the number of points each of them have.

If X wins, output X WINS BY n POINT(S), where is the difference between X and O's scores.

If O wins, output O WINS BY n POINT(S), where is the difference between O and X's scores.

Constraints

Sample Input 1 

5 5
XXXXX
OOXOO
OOXOX
XXOXO
XOOOX

Sample Output 1 

BOTH PLAYERS ARE TIED AT 2 POINT(S)

Sample Input 2 

4 4
XXXX
XXXX
XXXX
XXXX

Sample Output 2 

INVALID

Sample Input 3 

5 4
XOOO
XXXX
XOXO
XOOX
XOOO

Sample Output 3 

X WINS BY 4 POINT(S)

Explanation
In the first case, X scores two horizontal patterns in the first row. It can be seen after a thorough inspection, that these are the only "four-in-a-rows" of X. Similarly, O scores his two diagonals. Thus, the game is tied.

In the second case, the board is invalid since X played 16 times while O was not able to play at all. However, in a grid, both players should have played 8 times.

Finally, in the third case, X gets 4 points and O gets none. Thus, X wins by 4 points.
