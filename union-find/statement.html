When infatuated teenagers feel that they want to make their relationship official, they say that they are "mag-on". So, naturally, in social media, these young kids tell each other that "You and I - ON" to make things shorter. To make it even more brief, they would simply say "U. N. I. ON".

Joe wants to give his significant other a word search problem, containing 143 instances of "UNION", to signify his love for them. Joe wants to write the word search themselves but wants to make sure that he has exactly 143 instances in the end. And so, they enlist your help. Given a word search problem, can you find all the instances of "UNION" it has so that Joe knows how many instances to remove or add to arrive at exactly 143 instances?

The instances of "UNION" can appear written in any of the eight directions and may overlap with other instances.

Input Format

The first line of input contains a single integer, , the number of test cases.

The first line of each test case contains two integers separated by a single space, and , the number of rows and columns of the word search problem. The next lines each contain a string of uppercase letters, denoting the th row of the grid.

Output Format

For each test case, output a single line containing a single integer, the number of instances of "UNION" you can find.

Explanation

In the first case, a single instance of "UNION" appears backwards in the second row.

In the second case, there is an instance of "UNION" written horizontally in each of the five rows, plus two diagonal instances (down-right and up-right).
