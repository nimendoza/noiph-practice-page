At Lala Kindergarten, the students have lined up in two lines, one for boys and one for girls, for their flag ceremony. Lala Kindergarten has strict rules on diversity, and so there are exactly the same number of boys and girls. On a whim, each boy asked the girl beside him what their middle initial was. For some reason (maybe because they're kindergarteners?), each boy started to fight the girl beside him if they had different middle initials. Jonah, a teacher at Lala Kindergarten, wants to know how many kindergarteners will fight. Help her!

Input Format

The first line contains a single integer, , which is the number of boys at Lala Kindergarten (which is also equal to the number of girls at Lala Kindergarten).

The second line contains a single string composed of uppercase letters of length , representing the middle initials of each boy in Lala Kindergarten, in the same order that the boys are lined up for the flag ceremony.

The second line contains a single string composed of uppercase letters of length , representing the middle initials of each girl in Lala Kindergarten, in the same order that the girls are lined up for the flag ceremony.

Output Format

Output a single line containing a single integer: the number of fights that will happen at Lala Kindergarten because of their comparing middle initials.

Constraints

Sample Input 1 

2
AB
AZ

Sample Output 1 

1

Sample Input 2 

10
ABCDEFGJIJ
ABDDELLZZI

Sample Output 2 

6
