Cafe Sibuyas is located in a field with barbed wires and onions around it. Whenever they run out of stock, they ask their employees to find onions around the area. These employees are called the runners as they run around the area to get onions just in case they run out of stock.

Around the area of Cafe Sibuyas, there are barbed-wire fences that hinder the runners to get onions. Some runners can quickly remove a certain number of barbed-wire fences in order to access areas that they might not have been able to reach otherwise.

So far, there are two different skill levels for the runners - Skill Level 0 and Skill Level 1. The Skill Level indicates how many barbed-wire fences that employee can remove.

Given a skill level of a runner employee, what is the maximum number of onions they can gather?

Input Format

The first line of input contains , the number of test cases.

The first line of each test case contains two integers, and , separated by a space. The next lines each contains a string of length consisting of S, ., # and O.

  • . denotes a clear square area.
  • # denotes a square area with a barbed-wire fence.
  • O denotes a square area containing a single onion.
  • S denotes the square area in which Cafe Sibuyas is located.

It is guaranteed that S appears exactly once per test case.

Constraints

Output Format

For each test case, output a single line containing two integers separated by a space. The first integer is the maximum number of onions given the runner has Skill Level 0. The second integer is the maximum number of onions given the runner has Skill Level 1.
