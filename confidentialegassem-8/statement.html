Yuo is a new student in his class.

The girls in his class apparently communicate information using a confidential process to somehow jumble the message.

Each jumbled message has a corresponding sequence of integers attached to it: . To "unjumble" the message, look at the last letters of the jumbled message and reverse their order to get a new jumbled message. Then, look at the last letters of this new message and reverse their order to get a new jumbled message and so on. After the final reversal, you should get the real unjumbled message.

Carly wanted to send a super secret SMS to her classmate Rae. Late night, while watching television, she finally decided she will send the SMS to her friend. But instead of texting Rae, she accidentally texted Yuo. Her text message reads:

ireallyreallyreallyreallyreallyreallylikeyou
44 44 2 44 44

From:
Carly
09987765321

Unlike her mobile service provider, Carly doesn't seem to be smart. Especially when you know how she got in this position.

Now, Yuo can perfectly read the sentence but he knows that girls speak in code. Thus, he can't be too sure what it means. So, he wants to unjumble the message to be certain. Can you create a program that unjumbles messages such as these for Yuo?

Input Format

The first line of input contains an integer , the number of test cases.

Each test case consists of three lines. The first line contains two integers and , where is the length of the jumbled message (and of course, also the length of the unjumbled message). The second line contains a string , the jumbled message. It is guaranteed to contain exactly characters and it only consists of lowercase letters from a to z. The third line consists the sequence of integers.

Output Format

For each test case, output a single line containing a string: the unjumbled message.

Explanation

For the first test case we have 44 44 2 44 44.

Looking at the last 44 letters of ireallyreallyreallyreallyreallyreallylikeyou and reversing them, we obtain uoyekilyllaeryllaeryllaeryllaeryllaeryllaeri.

Looking at the last 44 letters of this new jumbled message and reversing them, we get ireallyreallyreallyreallyreallyreallylikeyou.

Reversing the last 2 letters (ou) of this new jumbled message and reversing them (into uo), we get ireallyreallyreallyreallyreallyreallylikeyuo.

Looking at the last 44 letters of this new jumbled message and reversing them, we get ouyekilyllaeryllaeryllaeryllaeryllaeryllaeri.

Looking at the last 44 letters of this new jumbled message and reversing them, we get ireallyreallyreallyreallyreallyreallylikeyuo, the unjumbled message.

Apparently, Carly really really really really really really likes Yuo! Awkward!
