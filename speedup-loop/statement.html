"Finding a flight using the website is too slow," says most of the customer survey sheets from your company. Your boss wants you to find improvements on the code:

printf("Trying to find you a flight...\n");
for (int i = 0; i < 1000000000; ++i) { ; }
printf("Flights found!\n");
list_flights();

The second line is what you call a speedup loop. If a developer can't find any bugs in the code but needs to present some improvement to the boss, you use the speedup loop. Here you remove one 0 from the longest string of 0s in the code. This will make the code around ten times faster! The developer can then report this speed up. Imagine all the time your customers will save with this little hack!

To save yourself some time in the future, you decide to automate the speedup process. Specifically, your task for this problem is to write a program that finds the longest string of 0 in some code and remove one of the 0. If there exists more than one longest string of 0 in the code, it should only act on the first one. If such a string cannot be found within the code, it should just add a comment at the top stating:
