ZOMGG!!! My dad’s going to take me to the Coldplay concert to see Chris Martin! He promised to give me VIP tickets! But, he has one condition. I need to be able to speak Tagalog sentences properly to be able to talk to people properly. W. T. F!!!!! Why Tagalog, Father??? So, anyway, I’m going to need some help. I know some Tagalog words but I usually fail in speaking proper Tagalog sentences. I try my best but I don't succeed! If I don't get to watch them, tears will come streaming down my face!!! Please help!!!!!!!

Input Format

Each line of input is a sentence you have to translate. We'll describe here the format of one sentence. Each sentence is of the following form:

[SUBJ] + " " + [MAKE] + " " + [VERB] + " the " + [NOUN] + [PUNC]

with [VERB] and [NOUN] being non-empty strings consisting only of letters A–Z and a–z.

Moreover, it is guaranteed that [VERB] has at least one vowel. Vowels are A, E, I, O, U, a, e, i, o, u. On the other hand, each subtask has different restrictions on each of [SUBJ], [MAKE], [NOUN] and [PUNC].

Constraints

There are at most lines of input, and at most characters per line of input.

Subtask 1 (6 points each): 

[SUBJ] = "I"
[MAKE] = "made"
[NOUN] = "ano"
[PUNC] = "."

i.e. all sentences are of the form "I made " + [VERB] + " the ano."

Subtask 2 (7 points each): 

[SUBJ] = "I" | "You" | "He" | "She" | "We" | "They"
[MAKE] = "made"
[NOUN] = "ano"
[PUNC] = "." | "!"

Subtask 3 (8 points each): 

[SUBJ] = "I" | "You" | "He" | "She" | "We" | "They"
[MAKE] = "made"
[NOUN] is unrestricted 
[PUNC] = "." | "!"

Subtask 4 (12 points each): 

[SUBJ] = "I" | "You" | "He" | "She" | "We" | "They"
[MAKE] = "make" | "makes" | "made"
[NOUN] is unrestricted 
[PUNC] = "." | "!"

Subtask 5 (17 points each): 

[SUBJ] = "I" | "You" | "He" | "She" | "We" | "They"
[MAKE] = "will make" | "make" | "makes" | "made"
[NOUN] is unrestricted 
[PUNC] = "." | "!"

Output Format

You must output a string of the following format:

[PANDIWA] + " " + [SIMUNO] + " ang " + [NOUN] + [PUNC]

where [SIMUNO] depends on the value of [SUBJ] as follows:

[SUBJ]    [SIMUNO]

 I         ko
 You       mo
 He        niya
 She       niya
 We        natin
 They      nila

and [PANDIWA] depends on [MAKE] as follows:

  • will make: Repeat the first syllable of [VERB]. If the last vowel of [VERB] is O or o, replace it with U or u, respectively. If the last letter is a vowel, add hin at the end. If the last letter is a consonant, add in at the end.
  • made: Add the string in before the first vowel.
  • make/makes: Repeat the first syllable then add the string in before the first vowel.

and [NOUN], [PUNC] are the same value as in the input. Moreover, the first syllable is defined to be everything before the first vowel, plus that vowel. Furthermore, if the first letter of the result is not capitalized, capitalize it.

Sample Input

I made tapon the ano.
He made pangako the tiket.
You make sulat the program.
I will make yakap the artista.

Sample Output

Tinapon ko ang ano.
Pinangako niya ang tiket.
Sinusulat mo ang program.
Yayakapin ko ang artista.
