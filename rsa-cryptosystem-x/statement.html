A cryptosystem is a combination of algorithms used to encrypt and decrypt data. One of the early cryptosystems was the RSA cryptosystem, developed by Ron Rivest, Adi Shamir and Leonard Adleman.

A good cryptosystem is one wherein an encrypted message can easily be decrypted ONLY by the intended recipient. Otherwise, it should be very difficult (or almost impossible) for an adversary to decrypt messages which are not meant for him. Most sensitive computer data are encrypted using this cryptosystem using large numbers.
