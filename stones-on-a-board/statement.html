Jeff has several differently colored stones (red, green, blue, yellow, and white). While playing checkers with his friend, Jared, they were struck by a question: "How many ways can we arrange all of Jeff's stones on this checkerboard?" After many hours of calculation, Jeff and Jared were able to calculate the number of ways to arrange the stones. However, now they have a new question in mind: "How many ways can we arrange all of Jeff's stones on a board with rows and columns?" Unfortunately, Jeff and Jared are very tired from their calculations, and need your help to answer their new question. Please write a program to satisfy their curiosity!

Note that you don't have to place a stone in every square. However, you cannot place more than one stone in any square.

Input Format

The first line contains two integers, and separated by a single space, which are the number of rows and columns (respectively) on the Jeff and Jared's board.

The second line contains five integers, , , , , and , which are the number of stones that Jeff has of each color.

Output Format

Output a single line containing a single integer: the last eight digits of the number of ways to arrange all of Jeff's stones on a board with rows and columns. If there are any leading zeros, remove them before displaying the result. For example, if the number of ways is 12345678900123456, then you should output 123456.

Constraints








Sample Input 1 

1 1
1 0 0 0 0

Sample Output 1 

1

Sample Input 2 

2 3
1 0 2 1 0

Sample Output 2 

180

Sample Input 3 

8 8
5 5 5 5 5

Sample Output 3 

55895040

Note
Displaying the last 8 digits of an answer is also called displaying the answer modulo . To understand this, you may want to read up on modular arithmetic.
