The Philippines, after having claimed the Spratly Islands, is now offering tours. These tours are open to everyone, including citizens from other countries who used to dispute the islands.

The government has decided that this is a good opportunity to show goodwill to neighboring countries, who might want to conduct research in the Spratly Islands. They couldn't possibly want anything else, like fishing, mining, oil and gas exploitation, or naming rights. Nope. Just research!

The islands are numbered , from north to south. Each of the islands has a north bridge, a bridge joining it to an island further north, and a south bridge, a bridge joining it to an island further south.

This makes bridges in total. The bridges are two-way bridges and can be crossed in either direction. Note that it is possible for two islands to have both a north bridge and a south bridge between them.

You are assigned to plan a tour that passes through all islands. The tour should start at an island, pass through several bridges, and end at an island, which can be the same island or a different island. To make the tour interesting, it should not pass the same bridge twice.

Given the above information, find an interesting tour through the Spratly Islands, or determine if there is no such tour. There may be multiple possible interesting tours: output any one.

Input Format

The first line of input contains , the number of test cases.

The first line of each test case contains , the number of islands.

The second line of each test case contains integers . This represents a north bridge from island to island .

The third line of each test case contains integers . This represents a south bridge from island to island .

for all

Output Format

For each test case, first output a single line containing YES or NO, denoting whether there exists at least one interesting tour. In addition, if you outputted YES, output two more lines describing one such tour. The first one must contain a single integer denoting the number of bridges in the tour. The second line must contain integers denoting the sequence of islands visited during the tour in order.
