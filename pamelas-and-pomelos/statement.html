The input contains exactly three lines. Each line contains a single integer, describing where each girl is with respect to the pomelo. The first line describes Pamela I's position.

Output Format
Output a single line containing the name of the person who is closest to the pomelo. Hence, that line will either contain PAMELA I, PAMELA II or PAMELA III.
